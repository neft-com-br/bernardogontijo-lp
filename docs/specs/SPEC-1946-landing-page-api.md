# SPEC: Landing Page API de Videochamadas

**Created**: 2025-12-04

## O Quê
Landing page em /lp para campanhas de ads pagos, focada em API de videochamadas para desenvolvedores e empresas que querem integrar videochamadas em suas aplicações.

## Por Quê
Converter tráfego de campanhas de ads pagos em cadastros, posicionar o videochamada.com.br como solução de API para desenvolvedores, e capturar leads B2B que buscam integrar videochamadas em suas plataformas.

## Comportamento Esperado
- Página acessível em videochamada.com.br/lp/
- Mantém Header e Footer padrão do site
- CTA principal "Criar conta grátis" redireciona para app.videochamada.com.br/cadastrar
- Estrutura com 6 seções principais: Hero com código exemplo, O Problema, A Solução, Diferenciais, Social proof/números, CTA final
- Mensagens focam em: API REST simples, 2.000 minutos grátis/mês sem cartão, preço em reais (R$ 0,015/min), SDKs oficiais (JavaScript, Python, PHP), webhooks, servidores no Brasil
