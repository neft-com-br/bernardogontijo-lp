# Bernardo Gontijo - Site Pessoal

Site pessoal construído com Next.js para geração de arquivos estáticos.

## Tecnologias Utilizadas

- [Next.js](https://nextjs.org/) - Framework React para geração de sites estáticos
- [TypeScript](https://www.typescriptlang.org/) - Superset tipado de JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitário
- [Framer Motion](https://www.framer.com/motion/) - Biblioteca de animações para React

## Desenvolvimento

```bash
npm install    # Instalar dependências
npm run dev    # Iniciar servidor de desenvolvimento
npm run build  # Gerar arquivos estáticos
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
